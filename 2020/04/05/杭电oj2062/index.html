杭电oj2062(sub sequence)

发表于 2020-04-05 更新于 2020-04-06 分类于 学习笔记 ， 算法题

杭电oj2062

hdu oj 第2062 题

本题有一定难度，刚开始比较没有头绪，想使用全排列的方法，将排列全部列出列，但是对于子集来说，是无法用字典序来全部列出的，而且时间复杂度也很大。参考了一下别人的思路。
当n=3时，我们知道按字典序排列有

  • {1} {1，2}  {1，3} {1，2，3} {1，3，2}
  • {2}  {2，1} {2，3} {2，1，3} {2，3，1}
  • {3}  {3，1} {3，2} {3，1，2} {3，2，1}

由此我们可以看出规律，An的集合必然可以分为n组，每一组中，去掉第一个单个子集，不难发现，剩下子集数量与An-1的子集数量相同，因此不难得出公式 f(n)=n*(f(n-1)+1),在计算出子集个数后。我们在进行讨论

  1. n=3时，m=10，num=15/3=5，index=(m-1)/num+1=2,rest=(m-1)%num+1=5,其中index代表m所在的组，rest代表第m个子集在该组所在位置，这里在第2组的第5位。此时我们输出组号为2，再在集合An中去掉2
    此时集合还剩下{1，3}，然后我们在判断，rest-1是否=0，如果等于0，说明其实在该组的第一个位置，也就是只要输出一个数就行了，如果rest-1不等于0，则在进行判断，使rest=rest-1，此时当我们输出了一个数后，容易发现，剩下要输出的数又相当于在An-1集合中输出第rest个子集，因此下一个循环使m=rest就可以。
  2. n=2时，m=rest=4. num=4/2=2，index=2，rest=2，继续输出

当n等于0，或是rest-1==0时停止循环。
代码如下：

#include<cstdio>
#include<iostream>
using namespace std;
int main(void){
    int n;
    __int64 m;
    __int64 a[21]={0};
    a[1]=1;
    for(int i=2;i<=20;i++){
        a[i]=i*(a[i-1]+1);
    }
    while(cin>>n>>m){
        int hashTable[21]={0};
        int N=n;
        __int64 num=a[n]/n;             
        __int64 index=(m-1)/num+1;      
        __int64 rest=(m-1)%num+1;      
        while(n!=0){
            int i=0;
            int count=0;
            for(int i=1;i<=N;i++){
                if(hashTable[i]==0){
                    count++;
                    if(count==index){
                        if(n==N){
                            cout<<i;
                        }else
                        {
                            printf(" %d",i);
                        }
                        
                        hashTable[i]=1;
                    }
                }
            }
            if(rest-1==0){
                break;
            }else{
                n--;
                num=a[n]/n;
                index=(rest-2)/num+1;
                rest=(rest-2)%num+1;

            }
        }
        cout<<endl;

    }
    return 0;
}

这道题还有一点小坑就是数组和m都要定义为64位整数
