组合数

组合数C(n,m)可以通公式计算C(n,m)=n!/(m!*(n-m)!)，如果只是单纯通过公式计算，在n为21的时候，n！就超出long long的数据范围了。这里介绍另外一种方法，即利用公式C(n,m)=C(n-1,m)+C(n-1,m-1)。这个公式可以离理解为从n个物体中选择m个的选择总数，就是先选最后一个，在剩下n-1个中选m-1个的次数，加上，不选最后一个，在剩下n-1个中选择m个的次数。通过这个递推式可以得到下列递归代码: 

long long C(long long n,long long m){
    if(m==0||m==n) return 1;
    return C(n-1,m)+C(n-1,m-1);
}

显然，用递归代码会比较简洁，但是会产生一个问题，就是重复计算，加大了很多时间，因此我们还可以利用数组将结果记录下来。 

long long res[67][67]={0};
long long C(long long n,long long m){
    if(m==0||m==n) return 1;
    if(res[n][m]!=0) return res[n][m];
    return res[n][m]=C(n-1,m)+C(n-1,m-1);
}

当然也可以利用循环来写。
