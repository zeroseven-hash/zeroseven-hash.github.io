  • Hexo是一个快速、简介且高效的博客框架。Hexo使用Markdown解析文章，在几秒内，即可利用靓丽的主题生成静态文章

一、搭建环境

1、安装Git环境

Git官方网址 https://git-scm.com/

有版本号输出即可

2、安装Nodejs

下载官网 https://nodejs.org/en/
直接下载即可

3、安装Hexo

  1. 在自己想要的位置新建文件夹
  2. 打开cmd，进入该文件夹，或直接右键，Git Bash Here
  3. 输入
  4. 安装完成后，输入 hexo -v 输出版本信息
  5. 输入 hexo init 初始化 会输出一下目录结构
  6. 输入npm install安装hexo所需要的依赖
  7. 输入 hexo g 生成页面
  8. 输入 hexo s 启动本地服务，这时候，启动浏览器，localhost：4000，则可访问页面。按住Ctrl+C结束服务
